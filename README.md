# minimalist tasbih app

During your "Vibe Coding" sessions, When you're waiting for your AI agent to finish writing lines of code, don't just stare at the screen—do some Tasbih. This ultra-minimalist, always-on-top counter is designed to keep you in the flow without cluttering your workspace.

## Features
- **Ultra-Small UI**: Takes up almost zero space on your screen.
- **Transparency**: 70% opacity so it blends perfectly with your editor.
- **Always-On-Top**: Stays visible even when you're switching between VS Code, Cursor, and Chrome.
- **Global Hotkeys**: Control everything without even focusing on the window.
- **Dynamic Width**: Automatically grows as your count increases.

## Quick Start
1. **Download**: download and extract `tasbih_app.zip`. https://github.com/ahmadali8186105/minimalist_dhikr_overlay/raw/refs/heads/main/tasbih_app.zip
2. **Run**: Double-click 'main.exe'. (No installation needed!)
3. **Persist**: The app remembers your count even if you close it.

## Global Shortcuts
| Action | Key Combination |
| :--- | :--- |
| **Increment Count** | `Ctrl + Alt + I` |
| **Reset to Zero** | `Ctrl + Alt + R` |
| **Minimize App** | `Ctrl + Alt + M` |
| **Restore / Show App** | `Ctrl + Alt + S` |
| **Close App** | `Ctrl + Alt + Q` |

## Customization
Want different shortcut keys?
- Run the app once to generate a `config.json` file.
- Open `config.json` in your editor and change the hotkeys.
- Example: Change `"<ctrl>+<alt>+i"` to `"<ctrl>+<shift>+p"`.

## Why use this?
In the world of Agentic Coding, we often have "wait states." Instead of checking social media or getting distracted, use those 20-30 seconds for Dhikr/Tasbih. It's a "plug and play" tool for the modern developer.
